# Discord Bot
**AUTHOR . . . . . . .** Shahar Ankonina                                                                              
**DESCRIPTION. . .** Discord music bot capable of controlling music in a discord voice chat

---
## About
A discord bot that acts as a music player on a discord call. Users can use / commands to play, queue, skip, stop, pause, or resume music. This allows a group of people to listen to the same songs at the same time.

---
## Features
- *Play:* Play (or automatically queue) audio.
- *Queue:* Saves audio to be played after the current one ends.
- *Skip:* Skips the current audio playing.
- *Stop:* Stops the bot entirely and causes it to quit the call.
- *Pause:* Pauses the audio currently playing.
- *Resume:* Resumes any paused audio.

## Installation
Follow these steps to run the project locally:

```bash
# Clone the repo
git clone https://github.com/AnkoninaShahar/DiscordBot.git

# Navigate into the directory
cd DiscordBot

# Install dependencies (Python example)
pip install -r requirements.txt

# Run the project
python discord_bot.py

```
